# SPEC-027: 連絡帳サマリ（保育園・幼稚園の世界観演出）

| 項目 | 内容 |
|---|---|
| 仕様ID | SPEC-027 |
| 機能名 | 連絡帳形式の日サマリ（保育園・幼稚園期） |
| 対応ファイル | `prototype/game.js` の `renderDaySummary()`, `buildRenrakuchoEntry()` / `prototype/index.html` の `#screen-day-summary` / `prototype/styles.css` の `.renrakucho-*` |
| 関連仕様 | SPEC-011, SPEC-012, SPEC-025, SPEC-028 |
| ステータス | Active（保育園のみ実装、幼稚園は Draft） |

## 1. 目的

1 日の終わりサマリ（S10、SPEC-025 §7.1）を **「連絡帳（先生 ⇔ 親 のやり取り）」風の見た目** に仕立てて、保育園・幼稚園期の世界観を強化する。

コンセプト：
- 数値やゲージで表現されない **成長の断片**（先生からのひと言、家庭のエピソード）を書面に載せる
- 定量的なパラメーター変動は **画面上部** に従来通りのゲージで示し、**内訳＝連絡帳** は画面中部以下に置く
- プレイヤーは「その日の出来事を親の視点で振り返る」疑似体験を得る

対象ライフステージ：
- **保育園**（1〜4歳）：実装対象
- **幼稚園**（5〜6歳）：Draft。後で実装
- **小学校以降**：連絡帳は廃止し、別の世界観演出（日記・部活ノート等）に切り替え予定

## 2. 用語定義

- **連絡帳**：その日 1 日を記述した手書き風の情報カード。`renrakucho-card` クラスで装飾
- **先生コメント（Teacher note）**：その日遊んだ内容に紐づくテンプレを 1 つランダム選出
- **家庭コメント（Parent note）**：プレイヤー（親）目線の近況を 1 つランダム選出
- **出席シール**：その日遊んだ遊びのアイコン列。左から時系列で並ぶ

## 3. 入力（Input）

- `player._daySnapshot`：S10 が参照する当日のスナップショット（SPEC-025 §7.1）
- 各 play の `renrakuchoTeacher` / `renrakuchoParent` テンプレ文（SPEC-028 §5.2）
- 当日の発見／新解禁（SPEC-011 の discoveries）

## 4. 出力（Output）

- S10 画面内に **上段＝パラメーター変動カード**、**下段＝連絡帳カード** の 2 段構成を描画
- 連絡帳カードは：日付 → 出席シール（遊びアイコン） → 先生から → 家庭から の順

## 5. 挙動（Behavior）

### 5.1 画面構成（S10 v2）

```
┌─────────────────────────────────┐
│  🌙 1日おわり                   │  タイトル
│  1日目 / 2歳 / 夏               │  期間

│  今日の遊び  📖×3 🛝×1           │
│  原体験（ゲージ）                │
│  スキル（ゲージ）                │
│  ステータス（体力・ジョウネツ）   │

│  │ YYYY年M月D日（曜日）   │       │
│  │ 出席シール 🛝 📖 🏖   │       │
│  │                      │       │
│  │ ✏️ 先生から         │       │
│  │ 「公園でお砂場遊び…」 │       │
│  │                      │       │
```

### 5.2 先生コメントの選び方

1. 当日遊んだ遊び（`_daySnapshot.playsById`）から **最も回数が多い遊び** を主役とする
2. その `renrakuchoTeacher` テンプレ（SPEC-028 §5.2）をランダムで 1 件選出
3. テンプレ末尾に、もし当日発見イベントがあれば「〇〇を見つけて目を輝かせていました」等を付け足す

欠落時のフォールバック：遊びに `renrakuchoTeacher` が未設定なら、一般的な「元気に過ごしていました☺」等の汎用コメントを使う。

### 5.3 家庭コメントの選び方

1. 遊びのうち **最もプレイヤーのジョウネツが伸びた主カテゴリ** に関連付けられたテンプレ
2. 同じ遊びでも `renrakuchoParent` をランダム抽選
3. 連日連続で同じテンプレを避ける（`_lastParentTemplateId` を保持して除外）

欠落時のフォールバック：「今日はよく眠れたみたい」等の汎用コメント。

### 5.4 ライフステージ判定

- `player.age` が 1〜4 のときのみ「保育園連絡帳」を表示
- `player.age` が 5〜6 のときは「幼稚園連絡帳」に切り替え（将来実装）
- それ以外は連絡帳は **非表示**。S10 は従来のパラメータ表示のみ

### 5.5 日付・曜日の擬似表現

- ゲーム内日数から擬似日付を算出：`初日 = 2026-04-01` とし、`day - 1` 日加算
- 曜日は `Date` オブジェクトから日本語表記を作る（月〜日）

## 6. UI への反映

- 画面：S10 日の終わりサマリ（`#screen-day-summary`）
- 新規 DOM：`#day-summary-renrakucho`（連絡帳カード）
- CSS：`.renrakucho-*` 系クラス（罫線、手書き風フォント、シール風アイコン）

## 7. 想定される利用シナリオ

### 7.1 砂場ばかりの 1 日
- 砂場 ×3
- 先生コメント：「公園でお砂場遊びをしました☺スコップを使って上手に穴を掘っていました。」
- 家庭コメント：「最近は公園の前を通るたび指をさして行きたいアピールをするようになりました。」
- 翌日、同じプレイヤーが砂場ばかり遊んだ場合は別テンプレを選出し重複を避ける

### 7.2 バラエティに富んだ 1 日
- 絵本 ×2、砂場 ×1、滑り台 ×1
- 主役＝絵本（最多）
- 先生「絵本を最後まで静かに聞けました」
- 家庭「うどんが大好きで昨晩は一玉完食しました」

## 8. 例外・エッジケース

- 当日遊びが 0 回（例：強制睡眠で丸 1 日寝込み）：連絡帳に「元気がないようで心配です」の汎用コメント
- 保育園休園日（将来）：「今日はおやすみしました」
