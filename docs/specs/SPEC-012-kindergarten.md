# SPEC-012: 幼稚園（4〜6歳）コアタイム

| 項目 | 内容 |
|---|---|
| 仕様ID | SPEC-012 |
| 機能名 | 幼稚園ライフステージ・コアタイム |
| 対応ファイル | （未実装） |
| 関連仕様 | SPEC-010, SPEC-011, SPEC-007 |
| ステータス | **Draft（未実装）** |
| 最終更新 | 2026-04-18 |

## 1. 目的

4〜6歳のプレイヤーは、平日 **09:00〜14:00** を幼稚園で過ごす。  
保育園よりもコアタイムは短いが、**集団生活の中で「文字が読めるようになる」「友達と遊ぶ方法を覚える」** といった、遊びの幅を拡げる学びが発生する。

ポイント：
- **友人数が増える速度が最大**のフェーズ
- **基礎パラメーター**（特に社会系・探求系）が底上げされる
- 保育園との違い：集団活動としての遊び（運動会、発表会など）が主軸になる

## 2. 用語定義

- **幼稚園コアタイム**：09:00〜14:00。
- **集団活動イベント**：遠足、運動会、発表会など年に数回発生する特別イベント。

## 3. 入力（Input）

- プレイヤー年齢 4〜6。
- 日付（`player.day`）。
- 季節（発表会・運動会は季節と連動）。

## 4. 出力（Output）

- `clockHour = 14` への進行。
- 原体験加算（social / explore / creative 中心）。
- 友人数増加（保育園より高確率）。
- 文字が読める状態フラグ `player.flags.canRead = true`（5歳到達時）。

## 5. 挙動（Behavior）

### 5.1 1日の流れ
```
09:00 登園、朝の会
10:00 クラス活動（文字、数字、歌）
11:00 外遊び or 工作
12:00 給食
13:00 自由遊び（園庭 or ホール）
14:00 降園
```

### 5.2 獲得値（確定分）
| 項目 | 値 |
|---|---|
| social | +5 |
| explore | +4 |
| creative | +3 |
| physical | +3 |
| ジョウネツ | +2 |
| 体力 | -12 |

### 5.3 発見プール（Draft）

| ID | テキスト | 効果 |
|---|---|---|
| `kg_read_letters` | ひらがなが読めるようになった！ | `flags.canRead = true`, 探求 +10, 新しい遊び「図鑑を読む」解禁 |
| `kg_new_friend` | クラスで一番仲良しのお友達ができた | 友人 +1, ジョウネツ +3 |
| `kg_group_tag` | クラスみんなで鬼ごっこの楽しさを知った | 社会 +5, 「クラス鬼ごっこ」解禁 |
| `kg_athletic` | 運動会で走るのが楽しかった | 身体 +8, 競技 +5 |
| `kg_stage` | 発表会で大きな声が出せた | 創作 +6, ジョウネツ +4 |
| `kg_share_toy` | おもちゃを貸してあげられた | 社会 +4 |

- 発見確率：75%（保育園より微増）。

### 5.4 解禁される遊び（Draft）

| 遊びID | 名前 | 解禁条件 |
|---|---|---|
| `read_book` | 図鑑を読む | `kg_read_letters` を発見 |
| `group_tag` | クラス鬼ごっこ | `kg_group_tag` を発見＋友人3人以上 |
| `stage_play` | 発表会ごっこ | `kg_stage` を発見 |

### 5.5 5歳イベント（マイルストーン）
- 5歳到達時、`flags.canRead = true` が未達成なら強制的に `kg_read_letters` を発火。
- 以降、全ての「読み物」系の遊びが発見プールに加わる。

## 6. UIへの反映
- S6 コアタイム画面を「幼稚園バージョン」として差し替え。
- HUD ラベル：「🏫 幼稚園」。

## 7. 想定される利用シナリオ
- 4歳・春：登園 → 集団活動 → 発見「ひらがなが読めるようになった」 → 帰宅後、S2 候補に「図鑑を読む」が追加。

## 8. 例外・エッジケース
- 4歳で途中入園した場合、1年間かけて友人数が自然増加（将来仕様）。
