## 1. 目的

4〜6歳のプレイヤーは、平日 **09:00〜14:00** を幼稚園で過ごす。  
保育園よりもコアタイムは短いが、**集団生活の中で「文字が読めるようになる」「友達と遊ぶ方法を覚える」** といった、遊びの幅を拡げる学びが発生する。

ポイント：
- **友人数が増える速度が最大**のフェーズ
- **基礎パラメーター**（特に社会系・探求系）が底上げされる
- 保育園との違い：集団活動としての遊び（運動会、発表会など）が主軸になる

## 2. 用語定義

- **幼稚園コアタイム**：09:00〜14:00。
- **集団活動イベント**：遠足、運動会、発表会など年に数回発生する特別イベント。

## 3. 入力（Input）

- プレイヤー年齢 4〜6。
- 日付（`player.day`）。
- 季節（発表会・運動会は季節と連動）。

## 4. 出力（Output）

- `clockHour = 14` への進行。
- 原体験加算（social / explore / creative 中心）。
- 友人数増加（保育園より高確率）。
- 文字が読める状態フラグ `player.flags.canRead = true`（5歳到達時）。

## 5. 挙動（Behavior）

### 5.1 1日の流れ
```
09:00 登園、朝の会
10:00 クラス活動（文字、数字、歌）
11:00 外遊び or 工作
12:00 給食
13:00 自由遊び（園庭 or ホール）
14:00 降園
```

### 5.2 獲得値（確定分）
| 項目 | 値 |
|---|---|
| social | +5 |
| explore | +4 |
| creative | +3 |
| physical | +3 |
| ジョウネツ | +2 |
| 体力 | -12 |

### 5.3 発見プール（Draft）

| ID | テキスト | 効果 |
|---|---|---|
| `kg_read_letters` | ひらがなが読めるようになった！ | `flags.canRead = true`, 探求 +10, 新しい遊び「図鑑を読む」解禁 |
| `kg_new_friend` | クラスで一番仲良しのお友達ができた | 友人 +1, ジョウネツ +3 |
| `kg_group_tag` | クラスみんなで鬼ごっこの楽しさを知った | 社会 +5, 「クラス鬼ごっこ」解禁 |
| `kg_athletic` | 運動会で走るのが楽しかった | 身体 +8, 競技 +5 |
| `kg_stage` | 発表会で大きな声が出せた | 創作 +6, ジョウネツ +4 |
| `kg_share_toy` | おもちゃを貸してあげられた | 社会 +4 |

- 発見確率：75%（保育園より微増）。

### 5.4 解禁される遊び（Draft）

| 遊びID | 名前 | 解禁条件 |
|---|---|---|
| `read_book` | 図鑑を読む | `kg_read_letters` を発見 |
| `group_tag` | クラス鬼ごっこ | `kg_group_tag` を発見＋友人3人以上 |
| `stage_play` | 発表会ごっこ | `kg_stage` を発見 |

### 5.5 5歳イベント（マイルストーン）
- 5歳到達時、`flags.canRead = true` が未達成なら強制的に `kg_read_letters` を発火。
- 以降、全ての「読み物」系の遊びが発見プールに加わる。

## 6. UIへの反映
- S6 コアタイム画面を「幼稚園バージョン」として差し替え。
- HUD ラベル：「🏫 幼稚園」。

## 7. 想定される利用シナリオ
- 4歳・春：登園 → 集団活動 → 発見「ひらがなが読めるようになった」 → 帰宅後、S2 候補に「図鑑を読む」が追加。

## 8. 例外・エッジケース
- 4歳で途中入園した場合、1年間かけて友人数が自然増加（将来仕様）。

## 10. 幼稚園「伸ばす」ギミック（Issue #14 反映 v2）

Issue #14 で幼稚園は **「伸ばす」楽しさ** と定義された。素養ポイントを貯めるフェーズから「投資する」フェーズへ切り替わる。

### 10.1 素養ポイントのデノミ（SPEC-037）
- 幼稚園入園時（age 4→5）に素養が **1/4 に切り詰められる**
- 詳細は SPEC-037 を参照

### 10.2 英霊システム解禁（SPEC-038）
- 幼稚園入園時に **英霊召喚画面** が開く
- 枠 1 つ。初期 5 体（パスカル / 福沢諭吉 / ダビデ / 孫尚香 / 雷電為右衛門）から選ぶ
- 召喚コストは **デノミ前** の素養から消費
- 詳細は SPEC-038 を参照

### 10.3 私立幼稚園への入園
- 入園時に **知性**・**身体** 等の数値が一定以上なら私立幼稚園が選択肢に入る
- 判定閾値（暫定）：
  - `soyou.intellect >= 80` または `soyou.body >= 80`
- 私立幼稚園：
  - コアタイムで得られる経験値が **+25%**
  - 行事が多い（毎月 1 つ追加の発見イベント）
  - 園児数が多いので友人が作りやすい（`kg_new_friend` の発生確率 +15%）

### 10.4 稽古教室（SPEC-039）
- 毎年 5 月の 1 日朝に「親からの習い事提案」モーダル
- 水泳・バイオリン・ピアノ・ダンス・絵画・英会話から 0〜2 個を選択
- 選んだ習い事は平日午後の 2h を占有する
- 3 ヶ月ごとに発表会系イベントが発生（SPEC-039 §5.3）

### 10.5 実装順序（ロードマップ）
1. SPEC-037 デノミ処理（1/4 切り詰め）
2. SPEC-038 英霊選択画面＋効果適用
3. SPEC-039 稽古教室の選択と稽古時間のコアタイム化
4. 私立幼稚園の選択分岐
