# SPEC-004: ランダムイベント

| 項目 | 内容 |
|---|---|
| 仕様ID | SPEC-004 |
| 機能名 | ランダムイベント |
| 対応ファイル | `prototype/game.js` の `EVENTS`, `rollEvent`, `renderEvent` |
| 関連仕様 | SPEC-003, SPEC-005, SPEC-007 |
| ステータス | Active |
| 最終更新 | 2026-04-18 |

## 1. 目的
遊び中に偶発的な出来事を発生させ、人生の不条理さや幸運を演出する。  
プレイヤーの意思決定に依存しないゆらぎによって、周回のリプレイ価値を生む。

## 2. 用語定義
- **イベント（Event）**：`EVENTS` 配列に定義された1件の出来事。
- **重み（weight）**：抽選時の出現確率の比率。
- **効果（effect）**：友人数・所持金・体力・ジョウネツ・原体験5カテゴリ に対する加減算。

## 3. 入力（Input）
- 遊びの描写フェーズ終了時の乱数（SPEC-003）
- プレイヤー状態（将来的には年齢・季節・カテゴリで候補を絞る）

## 4. 出力（Output）
- イベント画面（S4）での描写と、プレイヤー状態への加減算。

## 5. 挙動（Behavior）
### 5.1 発火条件
- 遊びの描写フェーズ後に、乱数 `r < 0.35` で発火。

### 5.2 抽選
- `EVENTS` から重み付き抽選で1件を選ぶ。
- 重みは各イベントの `weight`（未指定時は 1）。

### 5.3 効果の適用
| キー | 影響 |
|---|---|
| `friends` | 友人数（SPEC-007） |
| `money` | 所持金 |
| `stamina` | 体力 |
| `passion` | ジョウネツ |
| `physical` / `creative` / `explore` / `social` / `competitive` | 原体験（SPEC-005） |

- 適用値はクランプ（体力は 0〜100、友人数・所持金は 0 以上）。

### 5.4 代表的なイベント
| 名前 | 効果 | 重み |
|---|---|---|
| 近所の子に話しかけられた | 友人+1, 交流+5 | 10 |
| 100円玉を拾った | 所持金+100 | 6 |
| 先生に褒められた | ジョウネツ+5, 創作+5 | 8 |
| 派手に転んだ | 体力-10 | 6 |
| 天気が崩れた | 体力-5 | 5 |
| 没頭した | ジョウネツ+10 | 7 |

## 6. UIへの反映
- 画面：S4 ランダムイベント（`#screen-event`）
- 画面設計書：`docs/screen-design.md` §2.4

## 7. 想定される利用シナリオ
- ごっこ遊び中に「近所の子に話しかけられた」が発生 → 友人+1 + 交流+5。  
  そのまま S3 の結果フェーズに戻って、遊び本体の獲得値と合算表示される。

## 8. 例外・エッジケース
- 同じイベントが連続発生しても許容（プロトでは抑制しない）。
- 友人数がマイナスになるような効果は現状存在しない（将来追加時は SPEC-007 で検討）。
