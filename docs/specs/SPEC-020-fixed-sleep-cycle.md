# SPEC-020: 固定起床・就寝サイクル（低年齢）

| 項目 | 内容 |
|---|---|
| 仕様ID | SPEC-020 |
| 機能名 | 低年齢期の固定起床・就寝時刻と朝の遊び時間 |
| 対応ファイル | `prototype/game.js`（`WAKE_SLEEP_FIXED_AGES`, `getFixedSchedule`, `beginDay`, `goSleep`） |
| 関連仕様 | SPEC-006, SPEC-008, SPEC-010, SPEC-011, SPEC-019 |
| ステータス | Active（保育園分のみ実装）／他年齢はフレームワーク定義のみ |
| 最終更新 | 2026-04-18 |

## 1. 目的

幼少期のプレイヤーは自分で生活リズムをコントロールできない。  
**保育園〜小学校3年生（1〜9歳）は起床・就寝時刻を親に管理される** リアリティを再現する。

また、登園前の朝の時間を「自由に遊べる時間」として開放することで、  
**1日のリズム：朝の遊び → コアタイム → 放課後の遊び → 就寝** という構造を作る。

## 2. 用語定義

- **固定起床時刻（Fixed Wake Hour）**：ライフステージで強制される起床時刻。
- **固定就寝時刻（Fixed Sleep Hour）**：ライフステージで強制される就寝時刻。
- **朝の遊び時間（Morning Free Time）**：起床時刻〜コアタイム開始時刻の間のフリー時間。
- **夜の遊び時間（Evening Free Time）**：コアタイム終了時刻〜就寝時刻の間のフリー時間。
- **生活リズムコントロール不可フェーズ**：プレイヤーが就寝モードを選択できない年齢帯（1〜9歳）。

## 3. 入力（Input）

- プレイヤー年齢 `player.age`
- コアタイム設定（SPEC-010）
- 生活リズム `player.bioRhythm`（参考扱い。本仕様では使わない）

## 4. 出力（Output）

- 起床時刻の決定
- 就寝時刻の決定（＝強制就寝発火時刻）
- 朝の遊び時間＋夜の遊び時間の合計 `spareHours` の算出
- S5 就寝画面の **モード選択の可否**

## 5. 挙動（Behavior）

### 5.1 年齢別 固定時刻テーブル

| 年齢 | 起床時刻 | 就寝時刻 | コントロール可否 | 備考 |
|---|---|---|---|---|
| 1〜4 | 07:00 | 19:00 | **不可** | 保育園期 |
| 5〜6 | 07:00 | 20:00 | 不可 | 幼稚園期（Draft） |
| 7〜9 | 06:30 | 21:00 | 不可 | 小学校低学年（Draft） |
| 10〜12 | 06:30 | 22:00 | 一部可 | 小学校高学年、将来拡張 |
| 13〜 | — | — | 可（SPEC-006, SPEC-008 に従う） | |

> **実装範囲（プロト）**：保育園の 1〜4歳のみ。Active（実装）。  
> 5歳以降は Draft（フレームワークのみ）。SPEC-020 §5.1 の表を参照して将来実装する。

### 5.2 1日の時間割（保育園・1〜4歳の例）

```
07:00  固定起床
07:00 〜 09:00  朝の遊び時間（2h）← プレイヤーが遊びを選べる
09:00 〜 16:00  コアタイム（保育園 / SPEC-011）
16:00 〜 18:00  夜の遊び時間（2h）← プレイヤーが遊びを選べる
18:00 〜 19:00  食事・入浴バッファ（1h）
19:00  固定就寝
```

- プロトでは「朝 2h + 夜 2h = 合計 4h」を余剰時間として与える。
- 食事・入浴バッファの 1h は時計は進むが遊べない（便宜上、コアタイム終了〜就寝の間の `spareHours` を 2h に設定）。

### 5.3 起床処理
- **固定起床時刻** を `player.clockHour`, `player.clockMinute` に設定。
- 朝の余剰時間として、起床〜コアタイム開始までの時間を `player.spareHours` に設定。
  - 例：07:00 起床、コアタイム 09:00 → 朝の余剰 = 2h。
- コアタイム無し（老後）や 10歳以上の場合は本仕様は適用せず、SPEC-006 の生活リズムに従う。

### 5.4 コアタイム終了後の余剰時間
- コアタイム消化後、残余剰時間を `(固定就寝時刻 − 食事1h) − 現在時刻` で再計算。
- 例：16:00 コアタイム終了、19:00 就寝、食事1h → 余剰 = 18:00 − 16:00 = 2h。

### 5.5 強制就寝
- 夜の余剰時間が 0 になった瞬間、**就寝画面（S5）に自動遷移**。
- **1〜9歳はモード選択不可**：就寝画面では「早寝」「普通」「夜更かし」の選択肢を表示せず、  
  「ママが布団をかけてくれた」の演出のみ表示。自動的に `normal` 相当の回復処理を実行。
- 夜遊びの時間を使い切る前に「今日おわる」選択ができないわけではない。ただしプレイヤーの意志で就寝モードは選べない。

### 5.6 生活リズムパラメーターの扱い（低年齢期）
- 1〜9歳は起床・就寝が固定のため、`bioRhythm` はほぼ変動しない。
- 起床ごとに `bioRhythm` が 90 を下回っていれば 90 にリセット（親による矯正）。
- 昼寝の有無など、別の理由でリズムが変化するのは将来仕様。

### 5.7 体力との関係（SPEC-019 連携）
- 夜の遊び時間中に体力が 0 になった場合：
  - 1〜12歳 → 強制睡眠 2h → 余剰がまだあれば継続、0 なら就寝
  - 13歳以上 → 強制終了で即就寝（SPEC-020 §5.5 の自動 normal ルートを取る）

## 6. UIへの反映

- 起床画面（S1）の表示テキスト：
  - 「あなたは 2歳 / 夏の朝 07:00」
  - コンディション欄に「朝の遊び時間：2h」
- S5 就寝画面：
  - 1〜9歳は **モード選択カードを非表示**
  - 代わりに「もう19時。ママが布団をかけてくれた…」の演出と `[ おやすみ ]` ボタンのみ
- HUD：時刻表示は変わらず。余剰時間も自動計算後の値を表示。

## 7. 想定される利用シナリオ

### シナリオ：2歳の1日
- 07:00 起床（余剰 2h）
- 07:30 絵本を読む（-1h、余剰 1h）
- 08:30 ごっこ遊び（-1h、余剰 0h）
- 09:00 S6 保育園コアタイム → 16:00 帰宅（余剰 2h）
- 16:30 粘土遊び（-2h、余剰 0h）
- 19:00 自動就寝（モード選択なし）

## 8. 例外・エッジケース

- コアタイム開始時刻より起床が遅れた場合（将来の体調不良など）は、朝の遊び時間がゼロ or 負。
  - その場合、朝の遊び時間は 0 にクランプして S6 へ直行。
- 余剰時間がもともと 2h より短いステージ（幼稚園 09:00-14:00 後は 5h あるが、本仕様範囲外）。
- 休日の扱い：プロト段階では休日概念なし（毎日コアタイム発生）。
