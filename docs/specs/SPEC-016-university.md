## 1. 目的

19〜22歳のプレイヤーは **コアタイムを自分で設計できる** 唯一のライフステージ。  
大学の講義枠（09:00〜18:00）の中から、履修を選ぶことで週ごとに異なる時間配分ができる。  
**「時間は潤沢、カネは乏しい」** という本作最大の自由度フェーズ。

ポイント：
- **コアタイムを 1〜8 コマで自由選択**（1コマ = 90分）
- サークル / バイト / 一人暮らし / 免許取得 / 留学などの **長期投資** が可能
- 人生最大の「遊び時間」が取れるフェーズ

## 2. 用語定義

- **大学コアタイム（基準）**：09:00〜18:00（講義枠の上限）。
- **履修（Cirriculum）**：週の時間割。1コマ = 90分。最小1コマ〜最大16コマ/週。
- **単位（Credit）**：履修によって累積。卒業条件。

## 3. 入力（Input）

- プレイヤー年齢 19〜22。
- 週ごとの履修選択。
- サークル・バイト等の追加活動選択。

## 4. 出力（Output）

- 選んだコマ数だけ時刻が進む（9:00 + 90min × n）。
- 単位の加算、卒業フラグ。
- 大量の発見機会（講義、ゼミ、サークル、バイト先）。

## 5. 挙動（Behavior）

### 5.1 履修の選択 UI（Draft）
- 学期初めに、週の時間割を決定する「履修画面」を表示。
- 月〜金、1〜5限の 5×5=25 コマのうち、1〜16 コマを選ぶ。
- 選んだコマが毎日のコアタイムを構成する。

### 5.2 履修密度のトレードオフ
| 履修 | 時間 | 経験値 | 単位 | 備考 |
|---|---|---|---|---|
| 軽い（週4コマ） | ほぼフリー | 探求 +2/日 | 少 | 遊び時間最大。ただし留年リスク |
| 標準（週10コマ） | コアタイム半分 | 探求 +4/日 | 中 | バランス型 |
| 重い（週16コマ） | ほぼ埋まる | 探求 +8/日 | 多 | 早期卒業の可能性。遊び時間減 |

### 5.3 発見プール（Draft 抜粋）
| ID | テキスト | 効果 |
|---|---|---|
| `uni_circle` | 気の合うサークル仲間と出会った | 友人 +5, 社会 +10 |
| `uni_abroad` | 短期留学を決意した | 「海外旅行」解禁, 探求 +30 |
| `uni_license` | 運転免許を取得 | 「ドライブ」解禁 |
| `uni_lab` | 研究室で打ち込める分野が見つかった | ジョウネツ +20 |
| `uni_part_time` | 家庭教師バイトで稼げるようになった | 所持金 +30000/月 |

### 5.4 サークル・バイトは追加コアタイム
- コアタイム（講義）終了後、**任意で** サークル（2h）やバイト（4h）を積める。
- それぞれ独立の発見プールを持つ（本仕様書ではサマリのみ）。

### 5.5 卒業条件
- 22歳終了時点で、累積単位 ≥ 124 単位なら卒業。
- 不足した場合、留年 or 中退の分岐（将来仕様）。

## 6. UIへの反映
- 学期開始時：履修画面（新画面 S7 = 履修設定）
- 日常：S6 コアタイム画面を「大学」仕様に差し替え。
- HUD：「🎓 大学○年」

## 7. 想定される利用シナリオ
- 20歳・夏休み：履修 0 コマ → ほぼ全時間フリー → バイトで貯めた金で海外バックパッカー → 「海外旅行」と「英語」解禁。

## 8. 例外・エッジケース
- 一人暮らしモード（家賃発生）
- 就活期（4年次前期）：コアタイムが就活で埋まる

## 10. 大学「モラトリアム」ギミック（Issue #14 反映 v2）

Issue #14 で大学は **人生最大のモラトリアム期間** と定義。学業に励むも遊び呆けるも自由。

### 10.1 素養ポイントのデノミ（SPEC-037）
- 入学時（age 18→19）に素養が 1/4 に切り詰められる

### 10.2 有名校アドバンテージ
- 入学前の素養 / スキル / 受験結果（SPEC-015 §10.3）で進学先が決まる
- 有名校（MARCH 相当以上）に入ると：
  - **就活時期が早まる**（3 年後期 → 3 年前期）
  - **就活の書類選考通過率が +30%**
  - 「海外インターン」等の特別機会が解禁

### 10.3 学生時代に力を入れたこと（ガクチカ）
- ゲーム内で「力を入れた活動」を 1 つ選択できる（サークル／ゼミ／バイト／起業）
- 選択した活動を 2 年以上継続すると、就活時に +10%〜+30% の有利判定

### 10.4 落第システム
- 勉学を怠る＝授業（コアタイム）を 1 ヶ月以上全スキップすると、**単位不足で落第**
- 学年やり直しは時間ロスが痛い
- プレイヤーの判断で「サボる or 通う」を分刻みで選ぶ

### 10.5 資格取得
- **ロースクール**（法律）・**会計士学校**（会計）・**医学部**（別ルート）を選べる
- 資格取得に向けたコアタイムは重いが、就活が安定する
- 在学中 +2 年（ロー）／+1 年（会計士）の設定

### 10.6 就活
- 3 年生の夏 or 冬からエントリー開始
- 業界・企業選びは素養・スキル・ガクチカ・英霊（SPEC-038）によって決まる
- 選考結果で `player.career = {industry, company, position}` が確定

## 14. 就活の詳細フロー

### 14.1 大学 3 年夏：インターン（任意）
- 志望業界のインターンに応募
- 有名校は数も質も高め
- 参加すると業界スキル +5、選考突破率 +15%

### 14.2 大学 3 年秋〜冬：本エントリー
- 30〜100 社に応募（選択肢）
- エントリーシート通過：intellect × 大学ランク × 英霊効果
- 筆記試験：intellect + 関連スキル
- 1 次面接：social + sensitivity
- 最終面接：total + passion + 英霊

### 14.3 大学 4 年春：内定
- 複数内定の場合、プレイヤーが選択
- 選択時に `player.career` が確定
