## 1. 目的

19〜22歳のプレイヤーは **コアタイムを自分で設計できる** 唯一のライフステージ。  
大学の講義枠（09:00〜18:00）の中から、履修を選ぶことで週ごとに異なる時間配分ができる。  
**「時間は潤沢、カネは乏しい」** という本作最大の自由度フェーズ。

ポイント：
- **コアタイムを 1〜8 コマで自由選択**（1コマ = 90分）
- サークル / バイト / 一人暮らし / 免許取得 / 留学などの **長期投資** が可能
- 人生最大の「遊び時間」が取れるフェーズ

## 2. 用語定義

- **大学コアタイム（基準）**：09:00〜18:00（講義枠の上限）。
- **履修（Cirriculum）**：週の時間割。1コマ = 90分。最小1コマ〜最大16コマ/週。
- **単位（Credit）**：履修によって累積。卒業条件。

## 3. 入力（Input）

- プレイヤー年齢 19〜22。
- 週ごとの履修選択。
- サークル・バイト等の追加活動選択。

## 4. 出力（Output）

- 選んだコマ数だけ時刻が進む（9:00 + 90min × n）。
- 単位の加算、卒業フラグ。
- 大量の発見機会（講義、ゼミ、サークル、バイト先）。

## 5. 挙動（Behavior）

### 5.1 履修の選択 UI（Draft）
- 学期初めに、週の時間割を決定する「履修画面」を表示。
- 月〜金、1〜5限の 5×5=25 コマのうち、1〜16 コマを選ぶ。
- 選んだコマが毎日のコアタイムを構成する。

### 5.2 履修密度のトレードオフ
| 履修 | 時間 | 経験値 | 単位 | 備考 |
|---|---|---|---|---|
| 軽い（週4コマ） | ほぼフリー | 探求 +2/日 | 少 | 遊び時間最大。ただし留年リスク |
| 標準（週10コマ） | コアタイム半分 | 探求 +4/日 | 中 | バランス型 |
| 重い（週16コマ） | ほぼ埋まる | 探求 +8/日 | 多 | 早期卒業の可能性。遊び時間減 |

### 5.3 発見プール（Draft 抜粋）
| ID | テキスト | 効果 |
|---|---|---|
| `uni_circle` | 気の合うサークル仲間と出会った | 友人 +5, 社会 +10 |
| `uni_abroad` | 短期留学を決意した | 「海外旅行」解禁, 探求 +30 |
| `uni_license` | 運転免許を取得 | 「ドライブ」解禁 |
| `uni_lab` | 研究室で打ち込める分野が見つかった | ジョウネツ +20 |
| `uni_part_time` | 家庭教師バイトで稼げるようになった | 所持金 +30000/月 |

### 5.4 サークル・バイトは追加コアタイム
- コアタイム（講義）終了後、**任意で** サークル（2h）やバイト（4h）を積める。
- それぞれ独立の発見プールを持つ（本仕様書ではサマリのみ）。

### 5.5 卒業条件
- 22歳終了時点で、累積単位 ≥ 124 単位なら卒業。
- 不足した場合、留年 or 中退の分岐（将来仕様）。

## 6. UIへの反映
- 学期開始時：履修画面（新画面 S7 = 履修設定）
- 日常：S6 コアタイム画面を「大学」仕様に差し替え。
- HUD：「🎓 大学○年」

## 7. 想定される利用シナリオ
- 20歳・夏休み：履修 0 コマ → ほぼ全時間フリー → バイトで貯めた金で海外バックパッカー → 「海外旅行」と「英語」解禁。

## 8. 例外・エッジケース
- 一人暮らしモード（家賃発生）
- 就活期（4年次前期）：コアタイムが就活で埋まる
- 中退ルート
