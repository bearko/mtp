## 1. 目的

保育園期は素養を **貯める** ものだったが、幼稚園以降はそれを **使って投資する** 感覚に切り替える。歴代の偉人を「英霊」として呼び出し、自身に特殊効果を付与できるシステムを導入する。

Issue #14 の幼稚園セクションから起票。

## 2. 用語定義
- **英霊（Guardian Spirit）**：呼び出して効果を付与できる歴代偉人の存在
- **枠（Slot）**：英霊を同時に配置できる数。幼稚園で 1 枠、以降のステージで増える
- **召喚コスト（Summon Cost）**：英霊を呼ぶために消費する素養ポイント（複数カテゴリ）
- **交代（Swap）**：ライフステージ変更時に英霊を入れ替えできる

## 3. 前提
- 幼稚園（SPEC-012）から **解禁**
- 最初に呼び出せるのは **初期 5 体**。それ以降のステージで追加解禁（将来仕様）
- 召喚コストは **SPEC-037 のデノミ前** の素養から消費する

### 5.2 ステージ遷移時の挙動
1. 誕生日で新ステージに入る瞬間
2. **英霊召喚画面へ自動遷移**
3. プレイヤーは 0〜N 体を呼び出せる（枠の上限内）
4. 「決定」→ SPEC-037 デノミ実行 → 新ステージ 1 日目

プレイヤーが呼ばずに決定することもできる（スキップ）。

## 7. 効果適用

### 7.1 素養獲得バフ（soyouGain）
```js
function applyGuardianSoyouBuff(gain) {
  for (const g of player.guardians) {
    for (const [k, mul] of Object.entries(g.buffs.soyouGain || {})) {
      if (gain[k]) gain[k] = Math.round(gain[k] * (1 + mul));
    }
  }
  return gain;
}
```
`finalizePlay()` の `normalizeGainToSoyou()` 後に適用する。

### 7.2 スキル獲得バフ（skillGainCategory）
`finalizePlay()` のスキル経験値計算（SPEC-024 §5.3）で、カテゴリごとの倍率を加算：
```js
const cat = "reading";
let catMul = 1.0;
for (const g of player.guardians) {
  const b = (g.buffs.skillGainCategory || {})[cat];
  if (b) catMul += b;
}
skillExpPerCategory *= catMul;
```

### 7.3 特殊効果（例：私立小学校受験の成功率）
- 福沢諭吉を召喚中だと SPEC-013 私立小学校の合格判定 `passRate` が +10% される
- 実装は SPEC-013 側で参照（`player.guardians.find(g => g.id === "fukuzawa")`）

## 8. 失敗モード
- 素養ポイント不足：「呼び出す」ボタンを disabled に
- ステージが戻れる前提ではない：一度決定したら戻れない（保存で上書き）
- 枠が埋まっている時に新たに呼ぶ：既存の 1 体を解除するダイアログ

## 9. テスト観点
- 幼稚園 1 日目突入時に `#screen-guardian` が表示される
- パスカル召喚 → `player.guardians` に追加、intellect 80 消費
- 呼び出し後に SPEC-037 デノミが実行される（intellect 140 → (140-80) → 60 → /4 = 15）
- 絵本プレイ時の素養獲得量が通常 10 → パスカル効果で 12
- 福沢諭吉召喚中に私立小学校判定の加点が入る

## 12. 解除と交代

- ステージ遷移時に全枠リセット（もう一度選び直す）
- 同じ英霊を継続選択可能
- 「殿堂入り」制度：3 ステージ連続で召喚した英霊は、永続小効果付与（効果は 50% で永続）
