## 1. 目的

13〜15歳のプレイヤーは、平日 **08:00〜17:00** を中学校で過ごす。  
他のライフステージと比べて **コアタイムが最長** であり、余剰時間は少ない。  
その代わり、**部活動** を通じて友人が大幅に増え、**専門的な遊び** が解禁される。

ポイント：
- **部活動がコアタイムの延長**（〜18:00 まで延びる）
- 部活仲間は大人になっても残る長期的な友人
- スマホ解禁、SNS 解禁など「大人の遊び」の手前に触れる
- 思春期・反抗期の描写を入れる余地（ジョウネツの揺れ）

## 2. 用語定義

- **中学校コアタイム**：08:00〜17:00。
- **部活動延長**：放課後〜18:00（部活入部後のみ）。
- **文化祭・体育祭**：年間イベント。

## 3. 入力（Input）

- プレイヤー年齢 13〜15。
- 部活選択（13歳時点で1回だけ選ぶ）。

## 4. 出力（Output）

- 時刻進行：`clockHour = 17`（非部活日）または `18`（部活日）。
- 部活による専門経験値の加速。
- 友人数の安定増加。

## 10. 中学校「競う」ギミック（Issue #14 反映 v2）

Issue #14 で中学校は **「競う」遊び** と定義。他校と競争する構造、体力急上昇、夜更かし・恋愛が解禁される時期。

### 10.1 素養ポイントのデノミ（SPEC-037）
- 入学時（age 12→13）に素養が 1/4 に切り詰められる

### 10.2 部活動（Club Activity）
- **中 1 の 4 月** に部活選択モーダル（発表会／稽古教室と同じ UI フロー）
- 初期選択肢：運動部（サッカー／野球／バスケ／陸上／水泳）、文化部（吹奏楽／美術／茶道／合唱／科学）
- 部活は平日の **コアタイム後 15:30〜18:00 の 2.5h** を占有
- 毎月 / 学期ごとに **大会・コンテスト** イベントが発生（SPEC-039 の発表会システムを流用・拡張）
  - 大会結果で素養が大きく上下する
  - 優勝すると「地区大会」「全国大会」のネスト構造（学年進行で拡大）

### 10.3 体力上限の大幅アップ
- SPEC-019 の `staminaCap` を **40 → 60** に引き上げ
- 遊びの `staminaCost` もスケール（body 系は 10〜20 消費）

### 10.4 夜更かしの解禁
- SPEC-020 の固定睡眠サイクルから **プレイヤー選択制** に移行
- 22:00 就寝（通常）／ 24:00 就寝（夜更かし）／ 26:00 就寝（徹夜）の 3 択
- 夜更かしするとその晩 2〜4h の追加余剰時間を獲得するが、翌日のリズムが 15〜30 下がる（SPEC-006）

### 10.5 比較と自己肯定感
- 学力・体力・友人数に **周囲の平均値** が表示される
- プレイヤーの値と比較して、HUD 右下に小さく `偏差値 52` のようなインジケーター
- 偏差値は素養グレードとは独立した指標（将来拡張）

### 10.6 恋愛イベント（初恋）
- 中 1 夏〜高 3 の間、`social` と `sensitivity` が一定以上なら月 5% の確率で「気になる人ができた」モーダル
- 成立すると `player.relationships[]` にエントリが追加され、特別な遊び（デート）が解禁
- デート：映画／カラオケ／水族館（通常の遠出イベントに加えて、相手との組み合わせで成功判定）

### 10.7 実装順序
1. 部活選択画面＋稽古教室と同じ枠組みで運用
2. 大会イベントの成功率計算（SPEC-039 §5.3 を流用）
3. 夜更かし選択 → SPEC-020 の固定サイクルから動的サイクルへ移行
4. 偏差値計算と HUD への追加
5. 恋愛イベント（後回し）

## 13. 夜更かしの詳細システム

### 13.1 就寝時刻の選択（中学生から）
- 22:00（早寝）：リズム +5/日、stamina +15
- 24:00（普通夜更かし）：stamina +5、追加 2h 遊び可、リズム -10
- 02:00（本気の夜更かし）：追加 4h、リズム -25、翌日起床 10:00 以降

### 13.2 追加時間で出来る遊び（夜限定）
- `late_manga`：深夜に漫画を読む（sensitivity +15）
- `late_game`：夜にテレビゲーム（intellect +10, passion +10）
- `late_music`：夜に音楽を聴く／演奏（sensitivity +15）
- `late_study`：深夜勉強（intellect +20、ただし体力 -20）

### 13.3 累積ダメージ
- 連続 3 日夜更かしすると「体調不良」モーダル
- 1 週間続けると確率で感染症発動（SPEC-011 §15）
