## 1. 目的

13〜15歳のプレイヤーは、平日 **08:00〜17:00** を中学校で過ごす。  
他のライフステージと比べて **コアタイムが最長** であり、余剰時間は少ない。  
その代わり、**部活動** を通じて友人が大幅に増え、**専門的な遊び** が解禁される。

ポイント：
- **部活動がコアタイムの延長**（〜18:00 まで延びる）
- 部活仲間は大人になっても残る長期的な友人
- スマホ解禁、SNS 解禁など「大人の遊び」の手前に触れる
- 思春期・反抗期の描写を入れる余地（ジョウネツの揺れ）

## 2. 用語定義

- **中学校コアタイム**：08:00〜17:00。
- **部活動延長**：放課後〜18:00（部活入部後のみ）。
- **文化祭・体育祭**：年間イベント。

## 3. 入力（Input）

- プレイヤー年齢 13〜15。
- 部活選択（13歳時点で1回だけ選ぶ）。

## 4. 出力（Output）

- 時刻進行：`clockHour = 17`（非部活日）または `18`（部活日）。
- 部活による専門経験値の加速。
- 友人数の安定増加。

## 5. 挙動（Behavior）

### 5.1 部活動カテゴリ（Draft）
| 部 | 種別 | メイン経験値 | 解禁遊び |
|---|---|---|---|
| サッカー部 | 運動 | 身体 +10 / 競技 +8 | 試合観戦、朝練、合宿 |
| 吹奏楽部 | 文化 | 創作 +10 / 社会 +5 | 演奏会、楽器購入、音楽鑑賞 |
| 美術部 | 文化 | 創作 +12 | 展覧会、スケッチ旅行 |
| 科学部 | 文化 | 探求 +12 | 天体観測、実験 |
| 帰宅部 | なし | 全+2 | 放課後の自由時間が長い |

### 5.2 獲得値（通常日 / 確定分）
| 項目 | 値 |
|---|---|
| explore | +5 |
| social | +5 |
| physical | +3 |
| creative | +2 |
| ジョウネツ | +2 |
| 体力 | -15 |

部活日はさらに `-10` 体力追加。

### 5.3 発見プール（Draft 抜粋）
| ID | テキスト | 効果 |
|---|---|---|
| `jh_smartphone` | スマホデビューした | 「SNS」「動画視聴」解禁、社会 +5 |
| `jh_teacher_crush` | 尊敬できる先生に出会った | ジョウネツ +10 |
| `jh_club_friend` | 部活の同期と盛り上がった | 友人 +2, 社会 +8 |
| `jh_festival_stage` | 文化祭で発表した | 創作 +15 |
| `jh_exam_stress` | 定期テストで徹夜 | 体力 -10, リズム -5 |

### 5.4 3年間の流れ
- 1年：部活決定、初めての文化祭
- 2年：部活の中核メンバー、修学旅行
- 3年：引退試合、受験、卒業

## 6. UIへの反映
- S6 中学校画面：1日の流れに「部活動」「テスト」などを時系列で描写。
- HUD：「🎒 中学○年」

## 7. 想定される利用シナリオ
- 14歳・夏：吹奏楽部の合宿 → 友人+3, 創作+30, 「演奏会に行く」解禁。

- いじめイベント（ネガティブ発見、将来検討）
