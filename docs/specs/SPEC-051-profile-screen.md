## 1. 目的

パワプロ『マイライフ』における「シーズン成績の常時可視化」「特殊能力ファイル」を、本作のプロファイル画面として翻訳する。

- **Sensation の強化**：自分の通算記録・称号を誇れる場所を作る
- **Challenge の強化**：今のミッション進捗がひと目でわかる

## 2. 画面構成（S17）

ドックに新しく `📊 自分` アイコンを追加し、タップで遷移。

```

## 3. 4 つのタブの内容

### 3.1 🎯 今年のミッション（SPEC-050 連携）

```
📋 今年のミッション
  ─────────────────────
  ✅ 身体をのばそう
     💪 body 100 以上（達成！）
     報酬: 🏋 身体キッズ

  ⏳ 知性を育む
     🧠 intellect 100/120  [===== ] 83%

  ⏳ お友達マスター
     👥 2/3 人             [=====     ] 66%
```

進捗バーをアニメ付きで描画。達成済みは薄く、未達成はクッキリ。

### 3.2 🏆 称号（得意タグ）

```
🏆 称号コレクション（5/42）
  ─────────────────────
  ✨ 身体キッズ        [2歳 4月]
     効果：身体系遊びで経験値 +10%

  ✨ 公園の達人        [2歳 8月]
     効果：外遊び経験値 +15%

  ✨ 物知り博士        [3歳 3月]
     効果：知性系遊びで経験値 +10%

  🔒 読書マスター（絵本を 100 回で獲得）
  🔒 歌姫（お歌を 50 回で獲得）
  ...
```

- 獲得済みは色付きで表示、タップで詳細（獲得日時・効果・エピソード）
- 未獲得は 🔒 でシルエット、獲得条件のヒントを短文で

### 3.3 💫 ハマってる遊び TOP 5

```
💫 ハマってる遊び
  ─────────────────────
  1. 📖 絵本を読む       Lv 8（48 回）
  2. 🏖 砂場遊び         Lv 6（31 回）

  4. 🎵 お歌             Lv 3（12 回）

## 5. データ構造

```js
// player.profileStats を集約データとして保存
player.profileStats = {
  playCountByPlay: { picturebook: 48, sandbox: 31, slide: 24, ... },
  maxGrade: { body: "D", intellect: "C", sensitivity: "E", social: "F", passion: "E" },
  totalDaysPlayed: 360,
  totalPlayCount: 1247,
};
```

毎日の `finalizePlay()` で自動更新、プロファイル画面描画時に参照。

## 6. 称号（Titles）の自動認定ロジック

### 6.1 認定タイミング
- `finalizePlay()` 末尾と `nextDay()` 冒頭でチェック
- 条件を満たしたら `player.titles` に追加、同時にトースト通知

### 6.2 認定条件の例

```js
const TITLE_CONDITIONS = [
  { id: "picturebook_master", name: "絵本マスター",
    check: (p) => (p.profileStats.playCountByPlay.picturebook || 0) >= 100,
    bonus: { category: "reading", multiplier: 1.15 } },
  { id: "park_explorer", name: "公園の達人",
    check: (p) => {
      const park = ["near_park", "big_park"];
      return p.profileStats.totalPlayCountAt(park) >= 50;
    },
    bonus: { category: "outdoor_toy", multiplier: 1.15 } },
  { id: "all_soyou_F", name: "バランサー",
    check: (p) => SOYOU_KEYS.every(k => p.soyou[k] >= 20),
    bonus: { allSoyou: 1.05 } },
  // ... 42 種
];
```

### 6.3 バフの適用
`finalizePlay()` で該当タイトル所持時、`gainBoost` に乗算。
SPEC-047 §8.3 の持続バフ（`applyPersistBuffsToGain`）と同じ形式で追加。

## 7. UI の設計詳細

### 7.1 タブバー
