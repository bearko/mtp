## 1. 目的

23〜65歳のプレイヤーは、平日 **09:00〜18:00** を会社（または自営）で過ごす。  
カネは安定して入るが、**時間と体力は激減** する。  
本作のジレンマが最も強く現れるフェーズ。

仕事を通じた **学び・出会い・遊びの発見** を継続的に提供する。

ポイント：
- 所持金が安定して増える（月給）
- 体力消費が大きい
- 仕事関連の発見（出張、接待、昇進、転職、取引先との遊び）
- 結婚・育児との両立（結婚イベント以降は家族コアタイムも追加）
- 43年間という長期フェーズのため、**年代別の描写** が必要（20代 / 30代 / 40代 / 50代 / 60代）

## 2. 用語定義

- **社会人コアタイム**：09:00〜18:00（職業により残業あり）。
- **残業**：18:00〜22:00。疲労とカネのトレードオフ。
- **出張**：数日にまたがるコアタイム拡張。
- **育児コアタイム**：結婚・出産後、19:00〜21:00 に発生するサブコアタイム。

## 3. 入力（Input）

- プレイヤー年齢 23〜65。
- 職業選択（新卒時 1 回）。
- 結婚・育児フラグ。

## 4. 出力（Output）

- 時刻進行：通常 18:00、残業時は 19:00〜22:00。
- 所持金 +月給相当。
- 体力消費大。
- 年代別の発見。

## 5. 挙動（Behavior）

### 5.1 職業選択
| 職業 | 特徴 | 月給（Draft） |
|---|---|---|
| 会社員（事務） | コアタイム固定、残業可変 | ¥250,000 |
| 会社員（営業） | 出張多め、社外イベント豊富 | ¥280,000 |
| 技術職 | 集中モード、体力消費小 | ¥320,000 |
| クリエイティブ | 不規則、創作系経験値ブースト | ¥200,000〜 |
| 自営業 | コアタイム自由、収入リスク高 | 変動 |

### 5.2 年代別描写
| 年代 | 描写 |
|---|---|
| 20代 | 新人、同僚と飲み会、奨学金返済 |
| 30代 | 結婚、出産、マイホーム |
| 40代 | 管理職、子育て、親の介護 |
| 50代 | 単身赴任、趣味回帰 |
| 60代 | 役職定年、再雇用、定年準備 |

### 5.4 結婚・育児システム
- 結婚イベント：30代前半でランダム発生（条件：友人数 + 交流Lv が一定以上）
- 育児コアタイム：出産後 18 年間、19:00〜21:00 に発生
- 育児中は新しい発見プール（子どもとの遊び）が開く

### 5.5 退職・転職
- 自由に実行可能（プロト段階では未実装）。
- 転職で発見プールが切り替わる。

## 6. UIへの反映
- S6 社会人画面：仕事の1日＋残業選択＋帰宅後余剰時間の表示。
- HUD：「💼 社会人（XX職）」

## 7. 想定される利用シナリオ
- 28歳・技術職：週4日通常＋1日残業 → 月収＋40000 → 週末に「一人旅」解禁。
- 35歳・育児中：コアタイム 9-18 + 育児 19-21 → 余剰ほぼ0 → 子と一緒に遊ぶと二重に経験値が入る特例。

## 8. 例外・エッジケース
- 失業イベント（カネ激減、時間は増える）
- 病気イベント（休職、体力・リズム回復の場合も）

## 10. 社会人「培ってきた素養が職業を決める」（Issue #14 反映 v2）

Issue #14 で社会人は **「保育園からコツコツと培ってきた素養やスキルで業界や企業が決まる」** と定義。

### 10.1 素養ポイントのデノミ（SPEC-037）
- 就職時（age 22→23）に素養が 1/4 に切り詰められる

### 10.2 業界・企業判定
- 大学卒業時点の `player.soyou` / `player.skills` / ガクチカ / 英霊構成で、以下のような業界ルートが決まる：

| 条件 | 業界 |
|---|---|
| intellect A / 文字スキル Lv80+ | 出版・教育 |
| body A / 運動スキル Lv80+ / 体育系部活 2年 | スポーツ・健康 |
| sensitivity A / 音楽スキル Lv80+ | 芸術・エンタメ |
| social A / コミュ系スキル Lv80+ | 営業・コンサル |
| passion A / 挑戦系スキル | スタートアップ |
| 資格取得（ロー） | 法曹 |
| 資格取得（会計士） | 金融 |
| それ以外 | 一般企業 |

### 10.3 ライフイベント
- 結婚／出産／転職／管理職昇進／独立 などが 5 年おきに選択・発動
- それぞれで新しい遊び・制約が変化

### 10.4 長期シミュレーション
- 社会人期は age 23〜65 の 43 年間。1 日ずつプレイは現実的でないので、自動モード（SPEC-025）をデフォルトに推奨
- プレイヤーが気になる節目（結婚直前・定年直前）で手動に切り替えられる

## 11. 業界別キャリアツリー（9 業界）

### 11.1 IT・ソフトウェア業界
```
new_grad (25万) → junior (1年) → senior (3年) → tech_lead (5年) → manager (8年)
                                ↘ specialist (5年、技術特化)
                                     ↘ startup_cto (10年、独立)
```
- 素養要件：intellect A / 社交は任意
- 特徴：成長速度早い、転職で年収アップしやすい

### 11.2 金融業界
```
new_grad (28万) → junior (1年) → senior (3年) → 課長 (7年) → 部長 (12年) → 役員
                                ↘ 外資転職 (3年) → 年収 2x
```
- 素養要件：intellect + social A
- 特徴：夜勤・残業多し、リズム低下リスク

### 11.3 医療業界（医師）
```
研修医 (30万) → 後期研修 (3年) → 一般医師 (5年) → 専門医 (10年) → 開業 (15年)
```
- 素養要件：intellect SS / body A
- 特徴：長時間労働、社会的地位高

### 11.4 法曹業界（弁護士）
```
司法試験合格 → 修習 → 一般弁護士 (2年) → シニア弁護士 (5年) → パートナー (10年) → 独立事務所
```
- 素養要件：intellect SS / social A
- 特徴：最初は月 30 万、以降急上昇

### 11.5 教育業界
```
新任教師 (22万) → 一般教員 (5年) → 主任 (10年) → 教頭 (20年) → 校長 (30年)
```
- 素養要件：intellect A / sensitivity B+
- 特徴：安定、passion 必要

### 11.6 芸術・エンタメ業界
- 進路バラバラ：芸能活動／フリーランス／プロダクション所属
- 素養要件：sensitivity SS / passion A
- 特徴：年収変動大、最悪は半年収入 0 もあり

### 11.7 スポーツ業界
- 実業団・プロリーグ・指導者・解説者など
- 素養要件：body SS / passion A
- 特徴：引退早い（35-40歳）、セカンドキャリア必要

### 11.8 営業・商社業界
```
new_grad (24万) → 一般営業 (3年) → 主任 (7年) → 支店長 (15年) → 本部
```
- 素養要件：social A / intellect B+
- 特徴：成果主義、出張多し

### 11.9 一般事務
```
new_grad (22万) → 一般事務 (5年) → 主任 (10年) → 課長補佐 (20年) → 課長 (30年)
```
- 素養要件：任意（ボトムライン）
- 特徴：安定、残業少、passion は貯めにくい

## 13. 育児・介護フェーズの詳細

### 13.1 育児期（子どもの年齢別）

| 子どもの年齢 | プレイヤーの変化 |
|---|---|
| 0-1 歳 | 育児休暇可（社会人）3 ヶ月、仕事時間 -30% |
| 2-3 歳 | 保育園送迎、仕事時間 -20%、social +10 |
| 4-6 歳 | 幼稚園行事、仕事時間 -10%、sensitivity +10 |
| 7-12 歳 | 学費、塾選び、子と遊ぶ時間で passion +10 |
| 13-18 歳 | 反抗期対応、social 変動、受験サポート |
| 19+ | 独立、以降は仕送りか自立 |

### 13.2 介護期
- 親 age 85 以上で発動率 40% / 年
- 介護期：仕事時間 -30%、money -10万/月、passion ±0、sensitivity +15
- 介護期間：5-10 年
- 親の死別後 1 年は「喪失期」：passion -10/月

## 14. 40 年分の主要イベント

### 14.1 社会人 1-5 年目（23-27歳）
- 初配属・仕事覚え期
- 2 年目：新人飲み会
- 3 年目：初の昇給
- 4 年目：後輩の指導
- 5 年目：海外出張 or 社内交流会

### 14.2 6-10 年目（28-32歳）
- 結婚適齢期（SPEC-046）
- 転職チャンス多発
- 住宅購入検討
- 出産・育児

### 14.3 11-20 年目（33-42歳）
- 管理職候補
- 子育ての山場
- 親の健康問題
- 介護突入の可能性

### 14.4 21-30 年目（43-52歳）
- 管理職
- 健康診断の結果要注意
- 子ども進学で学費ピーク
- マイホームローン半ば

### 14.5 31-40 年目（53-62歳）
- 上級管理職 or 専門家
- 子ども独立
- 親の死別
- 定年予定

### 14.6 41-42 年目（63-65歳）
- 退職準備
- 再雇用 or 顧問
- 引退パーティー
- 老後へ
