## 1. 目的

23〜65歳のプレイヤーは、平日 **09:00〜18:00** を会社（または自営）で過ごす。  
カネは安定して入るが、**時間と体力は激減** する。  
本作のジレンマが最も強く現れるフェーズ。

仕事を通じた **学び・出会い・遊びの発見** を継続的に提供する。

ポイント：
- 所持金が安定して増える（月給）
- 体力消費が大きい
- 仕事関連の発見（出張、接待、昇進、転職、取引先との遊び）
- 結婚・育児との両立（結婚イベント以降は家族コアタイムも追加）
- 43年間という長期フェーズのため、**年代別の描写** が必要（20代 / 30代 / 40代 / 50代 / 60代）

## 2. 用語定義

- **社会人コアタイム**：09:00〜18:00（職業により残業あり）。
- **残業**：18:00〜22:00。疲労とカネのトレードオフ。
- **出張**：数日にまたがるコアタイム拡張。
- **育児コアタイム**：結婚・出産後、19:00〜21:00 に発生するサブコアタイム。

## 3. 入力（Input）

- プレイヤー年齢 23〜65。
- 職業選択（新卒時 1 回）。
- 結婚・育児フラグ。

## 4. 出力（Output）

- 時刻進行：通常 18:00、残業時は 19:00〜22:00。
- 所持金 +月給相当。
- 体力消費大。
- 年代別の発見。

## 5. 挙動（Behavior）

### 5.1 職業選択
| 職業 | 特徴 | 月給（Draft） |
|---|---|---|
| 会社員（事務） | コアタイム固定、残業可変 | ¥250,000 |
| 会社員（営業） | 出張多め、社外イベント豊富 | ¥280,000 |
| 技術職 | 集中モード、体力消費小 | ¥320,000 |
| クリエイティブ | 不規則、創作系経験値ブースト | ¥200,000〜 |
| 自営業 | コアタイム自由、収入リスク高 | 変動 |

### 5.2 年代別描写
| 年代 | 描写 |
|---|---|
| 20代 | 新人、同僚と飲み会、奨学金返済 |
| 30代 | 結婚、出産、マイホーム |
| 40代 | 管理職、子育て、親の介護 |
| 50代 | 単身赴任、趣味回帰 |
| 60代 | 役職定年、再雇用、定年準備 |

### 5.3 発見プール（Draft 抜粋）
| ID | テキスト | 効果 |
|---|---|---|
| `work_client_dinner` | 取引先と意気投合 | 友人 +1, 「高級レストラン」解禁 |
| `work_promotion` | 昇進した！ | 月給 +50,000, ジョウネツ +15 |
| `work_hobby_reignite` | 忘れていた趣味を再開した | 原体験系5つのうち一つに +20 |
| `work_burnout` | 燃え尽きた… | ジョウネツ -20, リズム -10 |
| `work_travel_biz` | 出張先で名物を楽しんだ | 探求 +10 |

### 5.4 結婚・育児システム
- 結婚イベント：30代前半でランダム発生（条件：友人数 + 交流Lv が一定以上）
- 育児コアタイム：出産後 18 年間、19:00〜21:00 に発生
- 育児中は新しい発見プール（子どもとの遊び）が開く

### 5.5 退職・転職
- 自由に実行可能（プロト段階では未実装）。
- 転職で発見プールが切り替わる。

## 6. UIへの反映
- S6 社会人画面：仕事の1日＋残業選択＋帰宅後余剰時間の表示。
- HUD：「💼 社会人（XX職）」

## 7. 想定される利用シナリオ
- 28歳・技術職：週4日通常＋1日残業 → 月収＋40000 → 週末に「一人旅」解禁。
- 35歳・育児中：コアタイム 9-18 + 育児 19-21 → 余剰ほぼ0 → 子と一緒に遊ぶと二重に経験値が入る特例。

## 8. 例外・エッジケース
- 失業イベント（カネ激減、時間は増える）
- 病気イベント（休職、体力・リズム回復の場合も）
