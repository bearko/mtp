## 1. 目的

ゲーム開始時のプレイヤーに対し、**操作・スキル・コアタイムを段階的に学ばせる** 最初の 2 週間を設計する。  
その間はスキップ機能（SPEC-025 §7.1.3）を封じ、基本サイクルに集中させる。

狙い：
- 初週：基本操作（遊びを選ぶ／1 日が進む／日サマリを見る）
- 2 週目：スキルで遊びの幅が広がる楽しさ＋コアタイム（保育園）の存在
- 3 週目以降：スキップ解禁で「人生をテンポよく進める」コアループへ移行

## 2. 用語定義

- **チュートリアルフェーズ（Phase）**：以下の 3 区分。
  - `phase0`（Day 1〜7, 初週）：保育園なし・**自動モード封印**・遊びは絵本→滑り台→砂場のみ段階解禁
  - `phase1`（Day 8〜14, 2 週目）：保育園開始、**情熱プロファイル選択 → 自動モード解禁**、スキル・発見・他の遊びを徐々に解禁（スキップはまだ）
  - `phase2`（Day 15〜）：**スキップ機能も解禁**、全機能開放
- **段階解禁（Gated unlock）**：初週に限り、プレイヤーが既に解禁済みの遊びだけドックに表示する制御
- **自動モード封印（Auto-lock）**：phase0 中は自動モードに切り替えられない（トグルが無効化 or 非表示、モード切替のアクションがすべて no-op）

## 3. 入力（Input）

- `player.day`（累計日数）
- `player.age`（2歳スタート想定）
- `player.unlockedPlays`（発見済みの遊びID配列）

## 4. 出力（Output）

- `tutorialPhase(day) -> "phase0" | "phase1" | "phase2"`
- 初週の S6 コアタイム画面の抑制（保育園なし）
- 初週のドック絞り込み（特定の遊びのみ許可）
- 2 週目以降、保育園を再開
- 3 週目以降、S10 の「⏩ スキップ」ボタンが解禁され、かつ **1 度だけ説明モーダル** を表示

## 5. 挙動（Behavior）

### 5.1 フェーズ判定

```
function tutorialPhase(day):
  if day <= 7:  return "phase0"    // 初週
  if day <= 14: return "phase1"    // 2週目
  return "phase2"                  // 3週目以降
```

### 5.2 Phase 0（初週 Day 1〜7）：保育園休業・遊びを段階解禁

- **保育園コアタイムを無効化**：`resolveCoreTime(age)` が phase0 中は `null` を返す（SPEC-010 §5.1 の外挿）
- **お昼寝時間**：13:00〜15:00 を "restTime" として扱い、この 2 時間は遊べない（時計は進む、体力は少し回復）
- **起床 07:00〜就寝 19:00 のうち、お昼寝以外は自由遊び** → 余剰時間 =（19-1-7）-（就寝前食事1）-（昼寝2） = 8h
- **遊びは初期 1 件から段階解禁**：
  - Day 1 開始時点で選べる遊びは `picturebook`（絵本を読む）**のみ**
  - 絵本を 1 回遊ぶと、絵本の中のエピソードで `slide`（滑り台）を発見 → 以降、ドックに絵本＋滑り台
  - 滑り台を 1 回遊ぶと、公園で `sandbox`（砂場遊び）を発見 → 以降、ドックに 3 種
- **その他の遊びは Day 8 までドックに出ない**（`unlockRequired` 的な扱いに phase0 限定フラグを追加）

#### 5.2.1 絵本→滑り台→砂場の解禁トリガー

```
発見 A: 絵本 1 回目のプレイ完了時に tut_discover_slide を発動
  - unlockPlayId: "slide"
  - メッセージ例: 「絵本に出てきた『すべりだい』をお家の近くで見つけた！」

発見 B: 滑り台 1 回目のプレイ完了時に tut_discover_sandbox を発動
  - unlockPlayId: "sandbox"
  - メッセージ例: 「お散歩中に公園の奥で『砂場』を見つけた！」
```

これらは通常の `NURSERY_DISCOVERIES` とは別の **チュートリアル専用発見**。
`data/events.json` に `scope: "tutorial_phase0"` として定義する（SPEC-028 §5.3 に拡張）。

**v2 改訂**：Phase 0 は保育園に通っていないので、文言は「保育園で見つけた」にしない。  
「お家の近くで」「お散歩中に」「絵本の中で」など、家庭内・親子生活のフレーバーに統一する。

### 5.3 Phase 1（2 週目 Day 8〜14）：保育園再開、自動モード解禁、発見拡大

#### 5.3.1 情熱プロファイル選択のタイミング変更（v2）
- **プロファイル選択は Phase 1 突入時（Day 8 起床直後）** に行う
- Phase 0 中はプロファイル不要（手動で絵本→滑り台→砂場を遊ぶだけ）
- Day 8 の起床モーダル（「月曜日から保育園！」）の直後に **S8 情熱プロファイル選択** を挿入
- プロファイル決定 → **自動モードがドック左端のトグルで使えるようになる**（それまでは封印）

#### 5.3.2 その他 Phase 1 の変更点
- `resolveCoreTime(age)` は通常どおり保育園を返す
- 保育園の発見プール（SPEC-011 §5.5）が解禁、`NURSERY_DISCOVERIES` がランダム発火
- 段階解禁された「絵本／滑り台／砂場」以外の遊び（粘土、鬼ごっこ、ごっこ遊び…）が **コアタイムの発見イベント** や **スキル Lv による解放** で徐々にドックに加わる
- S10 日サマリ（SPEC-027 連絡帳）は通常表示（先生コメント入り）
- スキップボタンはまだ非表示、代わりに「続ける」と「手動に切り替え」

### 5.4 Phase 2（3 週目〜）：スキップ解禁

- Day 15 の 1 回目の S10 表示時に **「スキップ機能が使えるようになった」の説明モーダル** を挿入（1 回のみ）
- 以降、S10 のフッターで「⏩ 週末までスキップ」「⏩ 明日の夜までスキップ」が **有効化**
- `isSkipUnlocked()` が以後常に `true` を返す
- チュートリアル関連の介入モーダルは以降出さない

### 5.5 スキップボタンの可否制御

```
function isSkipUnlocked():
  return tutorialPhase(player.day) === "phase2"
```

- S10 の 3 ボタンのうち、スキップ 2 つは `isSkipUnlocked()` が false のとき **非表示** とする
- かわりに `「このサマリを閉じる」` ボタンだけを表示（手動モードへ戻る導線のみ）
- `tutorial-hint` 要素に「スキップはあと◯日後から」のガイドを併置

### 5.6 既存仕様との相互作用

| 関連仕様 | 影響 |
|---|---|
| SPEC-010 コアタイム | phase0 では coreTime を null 返し、S6 を出さない |
| SPEC-011 保育園 | phase1 以降のみ実行。phase0 は休園扱い |
| SPEC-023 遊びツリー | phase0 では未解禁の遊びを `Future` ノードとして表示。ツリー内テキストで「解禁条件：◯◯を試すこと」を提示 |
| SPEC-025 ゲームテンポ | phase2 以降で §7.1.3 のスキップが有効化 |
| SPEC-027 連絡帳 | phase0 の連絡帳は「今週は慣らし保育でお休みでした」のテンプレを挿入 |

## 6. UI への反映

### 6.1 起床ヘッダーの拡張
- `tutorialPhase` に応じて **ヒントカード** を追加で 1 枚表示
  - phase0：「📖 絵本を読んでみよう」「🛝 絵本の中の滑り台を試してみよう」…（段階別）
  - phase1：「🏫 月曜日から保育園が始まります」「🎢 何か新しい遊びを見つけられるかも」
  - phase2：ヒント非表示

### 6.2 S10 フッターの制御
- `isSkipUnlocked()` が false → スキップボタン非表示、「このまま続ける」ボタンのみ

### 6.3 新画面 S11 チュートリアル説明モーダル（軽量）
- Day 15 到達時に 1 回だけ表示
- 内容：スキップボタンの使い方、1 週間 or 1 日単位で進められること
- 「わかった」ボタンで閉じる

## 7. 想定される利用シナリオ

### 7.1 Day 1 〜 3
- 07:00 起床 → 絵本アイコンだけドックに出現
- 絵本 × N 回、そのうち「絵本の中の滑り台を見つけた！」の発見（初回絵本後に必ず発火）
- 2 日目：絵本＋滑り台の 2 アイコン
- 滑り台プレイ後：「公園の砂場を見つけた！」発見
- 以降絵本・滑り台・砂場の 3 択で初週終了

### 7.2 Day 8
- 起床ヘッダーに「月曜日から保育園！」モーダル
- 保育園コアタイム開始、発見イベント解禁
- S10 連絡帳に初めて「先生から」が登場

### 7.3 Day 15
- S10 到達と同時に説明モーダル「スキップが使えるようになった」
- 以降 S10 フッターにスキップ 2 種が表示

## 8. 例外・エッジケース

- プレイヤーが絵本ばかり何回遊んでも滑り台解禁は **1 回目プレイ直後にのみ** 発火（2 回目以降の絵本では特別イベントなし）
- Day 1〜7 の間に体力ゼロ強制睡眠：お昼寝枠として扱い、S10 には「少しぐずぐずしていました」のテンプレ
- Day 7→Day 8 の切替は就寝→翌朝のタイミングでシームレスに
