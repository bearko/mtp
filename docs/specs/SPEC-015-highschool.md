## 1. 目的

16〜18歳は、平日 **09:00〜16:00** を高校で過ごす。  
中学校よりコアタイムは短くなり、**バイト解禁** によって所持金を自分で稼げるようになる。  
進路選択（進学・就職）のプレッシャーが、ジョウネツとの葛藤を生む。

ポイント：
- **バイト解禁**（16歳〜）：コアタイム後に追加のミニコアタイム枠として選択可能
- 友人関係が細分化（部活/クラス/バイト/SNS）
- 恋愛イベントの解禁
- 進路による分岐（大学・専門・就職）

## 2. 用語定義

- **高校コアタイム**：09:00〜16:00。
- **バイトコアタイム**：任意。`16:00〜20:00` のうち最大3時間を選択。
- **進路モード**：3年次に決定される分岐フラグ。

## 3. 入力（Input）

- プレイヤー年齢 16〜18。
- バイト有無（プレイヤー選択）。
- 進路選択（3年次）。

## 4. 出力（Output）

- 時刻進行：通常 16:00、バイト日は 20:00。
- 所持金の大幅増加（バイト）。
- 大学ステージへの接続（進学時）。

## 5. 挙動（Behavior）

### 5.1 1日の流れ
```
09:00 登校、朝のHR
10:00 授業
12:00 昼食（購買 / 弁当）
13:00 授業
16:00 放課後
[任意] 部活（〜18:00） or バイト（〜20:00）
```

### 5.2 獲得値（通常日 / 確定分）
| 項目 | 値 |
|---|---|
| explore | +4 |
| social | +4 |
| creative | +2 |
| physical | +2 |
| ジョウネツ | +2 |
| 体力 | -12 |

バイト日は +所持金（時給850〜1200円 × 時間）、体力 -10 追加。

### 5.3 進路モード分岐（18歳時）
| モード | 次ステージ | 特徴 |
|---|---|---|
| 大学進学 | SPEC-016 大学 | コアタイム自由化 |
| 専門・短大進学 | 大学ステージ（短縮版） | 2年で社会人へ |
| 就職 | SPEC-017 社会人（18歳〜） | カネが早く貯まるが、遊びの自由度で差がつく |

### 5.4 発見プール（Draft 抜粋）
| ID | テキスト | 効果 |
|---|---|---|
| `hs_first_love` | 初恋の相手と手紙を交換した | ジョウネツ +10 |
| `hs_part_time` | 初めてのバイトで給料を手にした | 所持金 +8000, ジョウネツ +8 |
| `hs_karaoke` | クラスでカラオケに行った | 「カラオケ」解禁, 社会 +5 |
| `hs_live` | 初めてのライブに行った | 「ライブ参戦」解禁, 創作 +10 |

## 6. UIへの反映
- S6 高校画面：コアタイム＋バイト選択UIを併設。
- HUD：「🎓 高校○年」

## 7. 想定される利用シナリオ
- 16歳・春：コアタイム → バイト選択（コンビニ3h）→ 20:00帰宅 → 所持金+3000 → 余剰1hで友人とカラオケ。

## 8. 例外・エッジケース
- 受験勉強中のバイト両立は体力・リズム消費が重い。
