## 1. 目的

16〜18歳は、平日 **09:00〜16:00** を高校で過ごす。  
中学校よりコアタイムは短くなり、**バイト解禁** によって所持金を自分で稼げるようになる。  
進路選択（進学・就職）のプレッシャーが、ジョウネツとの葛藤を生む。

ポイント：
- **バイト解禁**（16歳〜）：コアタイム後に追加のミニコアタイム枠として選択可能
- 友人関係が細分化（部活/クラス/バイト/SNS）
- 恋愛イベントの解禁
- 進路による分岐（大学・専門・就職）

## 2. 用語定義

- **高校コアタイム**：09:00〜16:00。
- **バイトコアタイム**：任意。`16:00〜20:00` のうち最大3時間を選択。
- **進路モード**：3年次に決定される分岐フラグ。

## 3. 入力（Input）

- プレイヤー年齢 16〜18。
- バイト有無（プレイヤー選択）。
- 進路選択（3年次）。

## 4. 出力（Output）

- 時刻進行：通常 16:00、バイト日は 20:00。
- 所持金の大幅増加（バイト）。
- 大学ステージへの接続（進学時）。

## 5. 挙動（Behavior）

### 5.1 1日の流れ
```
09:00 登校、朝のHR
10:00 授業
12:00 昼食（購買 / 弁当）
13:00 授業
16:00 放課後
[任意] 部活（〜18:00） or バイト（〜20:00）
```

### 5.2 獲得値（通常日 / 確定分）
| 項目 | 値 |
|---|---|
| explore | +4 |
| social | +4 |
| creative | +2 |
| physical | +2 |
| ジョウネツ | +2 |
| 体力 | -12 |

バイト日は +所持金（時給850〜1200円 × 時間）、体力 -10 追加。

### 5.3 進路モード分岐（18歳時）
| モード | 次ステージ | 特徴 |
|---|---|---|
| 大学進学 | SPEC-016 大学 | コアタイム自由化 |

| 就職 | SPEC-017 社会人（18歳〜） | カネが早く貯まるが、遊びの自由度で差がつく |

## 10. 高校「見据える」ギミック（Issue #14 反映 v2）

Issue #14 で高校は **「見据える」遊び** と定義。受験か非受験かの選択、将来の進路を真剣に考える時期。

### 10.1 素養ポイントのデノミ（SPEC-037）
- 入学時（age 15→16）に素養が 1/4 に切り詰められる

### 10.2 進路選択（高 2 の 10 月）
- 「大学受験する／しない」モーダル
  - 受験する：コアタイム後の塾通い必須、部活とバイトは制限
  - しない：そのまま就職 or 専門学校進路。遊びの自由度は高い

### 10.3 受験モード
- 高 3 の 4 月以降：`player.examMode = true`
- 平日は 塾 18:00〜22:00 でほぼ自由時間なし
- 日曜日は模試（月 1 回）
- 模試の結果次第で `player.examStanding` が更新され、大学合格判定に反映

### 10.4 バイト（SPEC-015 §5 既存）
- 受験生はバイト禁止（または週 1 に制限）
- 非受験生は週 3〜5 回まで可能

### 10.5 学園祭・修学旅行・体育祭
- 年に 1 回、季節固定イベント
- 修学旅行（2 年生の秋）は 3〜4 日消化し、social +50 / sensitivity +30 / 新しい友人 +3

### 10.6 将来設計の可視化
- S2 画面の上部に「あと N 日で高校卒業」「志望校：○○」の小さなチップ
- プレイヤーが「見据える」感覚を持てるよう設計

## 11. 偏差値システム

### 11.1 模試の実施
- 高 2 の 10 月から毎月 1 回、模試が開催
- 教科：国語・数学・英語・理科・社会 の 5 科目
- 各教科の偏差値：`intellect` と 関連スキル（SPEC-024）の累計で算出

### 11.2 偏差値算出式
```js
function calcDeviationValue(subject) {
  const baseIntellect = player.soyou.intellect;
  const skillSum = (player.skills[subject] || {lv:1}).lv * 5;
  const juku = player.jukuAttendancePct || 0; // 塾通い率 0-1
  const rawScore = baseIntellect + skillSum + juku * 30;
  // 母集団は全国 100 万人想定、平均 100、標準偏差 20 で偏差値化
  return Math.round(((rawScore - 100) / 20) * 10 + 50);
}
```

### 11.3 偏差値帯と大学ランク

| 偏差値 | 大学ランク | 該当校例 |
|---|---|---|
| 70+ | S（超一流） | 国立トップ、慶早、医学部 |
| 65-69 | A（上位） | 国立主要、私立トップ |
| 58-64 | B（中堅） | 地方国立、私立中堅 |
| 50-57 | C（一般） | 一般私立 |
| 45-49 | D（下位） | 地方私立 |
| <45 | 浪人 or 専門 | – |

## 14. 受験モードの詳細フロー

### 14.1 高 2 の 10 月：進路選択モーダル
- 「大学受験する？」Yes/No
- Yes：受験モード開始、予備校通い選択
- No：就職活動モードに移行

### 14.2 予備校通い（受験生のみ）
- 選択肢：3 校
  - `prep_yoyo`：大手予備校（月 80k、intellect +10/回、偏差値 +3/月）
  - `prep_small`：地元小予備校（月 40k、intellect +5/回）
  - `prep_online`：オンライン講義（月 20k、intellect +3/回、自己管理で変動）

### 14.3 高 3 の受験期（1-3 月）
- 1 月：センター試験相当（共通テスト）
  - SPEC-043 評価、5 科目それぞれ判定
- 2 月：私立大学入試（複数校）
- 2 月末-3 月：国立大学前期
- 3 月後半：国立後期
- 合否発表：3 月末

### 14.4 浪人の選択
- 不合格で全滅した場合「浪人する？」モーダル
- 浪人：+1 年、予備校コアタイム必須、intellect 確実に伸びるが体力減
- 2 浪まで可能、3 浪は諦めて専門 or 就職
