## 1. 目的

66歳以降、プレイヤーは **コアタイムから完全に解放** される。  
時間は潤沢、カネは年金と貯蓄で安定、だが **体力・健康** が主な制約となる。

長く遊び続けられた分だけ、蓄積された原体験とジョウネツが結実する。  
**「結局あなたはなにで遊んだ人生だったのか」** が浮かび上がるフェーズ。

ポイント：
- コアタイム無し（`coreTime = null`）
- 再就職・ボランティア・孫との遊び・趣味の深化
- 体力減衰で一部の遊びが徐々にロックされる（登山、長旅行など）
- 100歳到達時エンディング

## 2. 用語定義

- **年金**：毎月の固定収入（Draft: ¥150,000）。
- **体力減衰**：年齢に応じて `stamina` の最大値が下がる。
- **健康度（Health）**：`stamina` とは別の隠しパラメーター（将来）。

## 3. 入力（Input）

- プレイヤー年齢 66〜100。
- 蓄積されてきたパラメーター（これまでの人生のすべて）。

## 4. 出力（Output）

- コアタイムなしで、S1 → S2 へ直行。
- 年金入金（月初）。
- 年齢進行による最大体力減衰。

## 5. 挙動（Behavior）

### 5.1 ライフステージの細分（Draft）
| 年齢 | 呼称 | 特徴 |
|---|---|---|
| 66〜75 | 前期高齢 | 元気、旅行、趣味の時間 |
| 76〜85 | 中期高齢 | 体力減衰、近場中心 |
| 86〜100 | 後期高齢 | 室内遊び中心、回想 |

### 5.2 体力上限の減衰
```
max_stamina = max(30, 100 − (age − 65) × 1.5)
```
- 66歳：98.5
- 80歳：77.5
- 100歳：47.5

### 5.3 遊びのロック条件
- 「登山」「海外長期旅行」「スポーツ大会」など体力上限 80 未満で徐々にロック。

### 5.4 発見プール（Draft 抜粋）
| ID | テキスト | 効果 |
|---|---|---|
| `old_grandchild` | 孫が遊びに来た | 友人 +1, ジョウネツ +15 |
| `old_garden` | 家庭菜園で新品種に挑戦 | 探求 +10 |
| `old_memoir` | 若い頃の日記を読み返した | **過去の原体験が +5% ジョウネツとして再結晶** |
| `old_reunion` | 同窓会で旧友と再会 | 友人 +3, 社会 +10 |

### 5.5 エンディング（100歳）
- 100歳の誕生日に **エンディング画面** へ遷移。
- 総ジョウネツ、原体験カテゴリLv、遊んだ種類数、解禁総数、家族・友人数などを表示。
- 「あなたの人生は○○な人生でした」のテキスト生成（Draft）。

## 6. UIへの反映
- S6 コアタイム画面は **表示しない**（`coreTime = null`）。
- HUD：「🪑 老後（○○歳）」

## 7. 想定される利用シナリオ
- 70歳：年金15万 / 体力90 / 家庭菜園Lv8 → 毎日「菜園」「読書」「孫と遊ぶ」の繰り返し → ジョウネツが最高潮に。
- 95歳：体力50で「散歩」「囲碁」「回想」のみ → 発見 `old_memoir` で、若い頃の経験が再結晶。

## 8. 例外・エッジケース
- 寿命変動（運パラメーターで ±10年）
- 病気イベント（入院で余剰時間減）
- 老人ホーム入居
