## 1. 目的

プレイヤーの成長に応じて、ライフステージの「学校選び」という重大な分岐を作る。Issue #14 の幼稚園以降のすべてに登場する「私立 vs 公立」「一般 vs 有名校」の選択肢を統一フレームワークで扱う。

## 2. 用語定義
- **進路選択（Admission Choice）**：親（プレイヤー）が学校を選ぶ／受験する／受験しない
- **入試（Entrance Exam）**：合否判定される試験。SPEC-043 の evaluation-event を継承
- **ランク**：学校の難易度・ブランド。S（超一流）／A（名門）／B（中堅）／C（標準）／D（底辺）
- **通学スケジュール**：選択した学校に応じたコアタイムの時間帯・日数

## 3. 入試のタイミング

| 入試 | 対象年齢 | 発生タイミング | 備考 |
|---|---|---|---|
| 私立幼稚園受験 | 4歳 | 保育園 → 幼稚園移行時 | 任意、通常は公立で OK |
| 私立小学校受験 | 6歳 | 幼稚園 → 小学校移行時 | 任意、通常は公立で OK |
| 中学受験 | 11-12歳 | 小学 6 年の 2 月 | 希望者のみ、公立中は無条件進学 |
| 高校受験 | 15歳 | 中学 3 年の 2 月 | 全員受ける |
| 大学受験 | 17-18歳 | 高校 3 年の 1-3 月 | 希望者のみ、浪人あり |

## 5. 受験フロー（共通）

### 5.1 準備期
1. ライフステージの所定時期（§3）に **進路モーダル** が出る
2. プレイヤーは：
   - 受験する（学校ランクを選択）
   - 受験しない（次のステージは公立）
3. 受験すると、受験期間中は **塾・予備校の追加コアタイム** が強制される

### 5.2 実施期（受験当日）
- 2 月 or 3 月 の所定日に `runEntranceExam(schoolRank)` 発火
- SPEC-043 の評価イベントを継承して合否を判定

### 5.3 判定式
```js
function calcAdmissionResult(schoolRank, subject) {
  const line = schoolRank.passLine[subject]; // 例: intellect 140
  const actual = player.soyou[subject];
  const ratio = actual / line;
  // 偏差値補正
  const dev = (player.examStanding || 50) / 50;
  const finalScore = ratio * dev;
  // 英霊補正
  if (schoolRank.guardianBonus && player.guardians.some(g => g.id === schoolRank.guardianBonus)) {
    return finalScore * 1.1;
  }
  return finalScore;
}
```

### 5.4 合否
- `finalScore >= 1.0` → 合格
- `0.9 ≤ finalScore < 1.0` → 補欠（10% の確率で合格）
- `finalScore < 0.9` → 不合格

### 5.5 失敗時
- 幼稚園／小学校：公立 or 下位ランクに自動進学
- 中学受験：公立中学に進学
- 高校受験：**滑り止め** の B or C ランクに合格していればそこに進学
- 大学受験：浪人（+1 年・再受験）or 専門学校

## 6. 学費と連動（SPEC-042）

- 私立は学費が高い（幼稚園 30k/月、小学校 80k/月、中学校 60k/月、高校 100万/年、大学 120万/年）
- 親の収入（`player.parentIncome`、初期は仮値）で払えない場合、選択不可 or 奨学金
- 奨学金（SPEC-未定）は将来実装。プレイヤーが返済することになる

## 7. エスカレーター式・付属校

- 私立幼稚園 S ランク → 系列小学校・中学校へ内部進学（受験なし）
- 付属大学への優先入学
- 内部進学の可否は関係値（SPEC-041）と成績で判定

## 8. プレイヤー体験（感情設計）

- 受験前：緊張感、塾でのコアタイム拘束
- 受験当日：特別な演出（講堂の雰囲気、時計、他の受験生）
- 結果発表：合格 or 不合格で大きく変わる演出
- 合格：「新しい世界が開ける」感覚
- 不合格：悔しさ + 次へのリベンジ意欲（passion +10）

## 9. テスト観点
- 中学受験 S ランク合格ライン（intellect 140 / 偏差値 70）で合格確率 90% 以上
- 英霊福沢を召喚中だと合格判定に +10%
- リズム絶不調時の受験は成功率が -15%
- 浪人ルート：1 年追加で再受験可能、2 浪まで
